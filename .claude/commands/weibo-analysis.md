# 微博热搜分析 (/weibo-analysis)

## 描述
斜杠命令：自动分析微博热搜数据，获取背景信息，生成深度洞察和产品创意报告

## 触发方式
输入 `/weibo-analysis` 或包含以下关键词的消息：
- `weibo`
- `微博热搜`
- `热搜分析`
- `微博`
- `热点分析`

## 执行流程
```
用户输入 /weibo-analysis
↓
调用 weibo_analysis_command.py
↓
1. 获取微博热搜数据 (微博API/天API)
2. 深度分析前10个话题
3. 使用百度AI搜索API获取背景信息
4. 生成话题分类、洞察和产品创意
5. 输出HTML报告和JSON数据
```

## 功能特性
✅ **自动数据获取** - 实时获取微博热搜榜单
✅ **背景深度调研** - 使用百度千帆AI搜索每个话题
✅ **智能话题分类** - 自动识别娱乐、社会、经济、科技等分类
✅ **产品创意生成** - 每个话题提供2个软件产品方案
✅ **可视化报告** - 生成美观的HTML分析报告
✅ **JSON数据输出** - 保存结构化分析结果

## API配置
- **百度千帆AI搜索API**: 获取话题背景信息
- **微博热搜API**: 获取实时热搜数据
- **内置分类算法**: 智能话题分类

## 输出文件
1. `weibo_analysis_report_YYYYMMDD.html` - **主报告** (可视化HTML)
2. `weibo_analysis_results_YYYYMMDD_HHMM.json` - 结构化数据
3. `weibo_hot_data_YYYYMMDD_HHMM.json` - 原始热搜数据

## 每个话题的分析结构
```
话题: [热搜标题]
├─ 📊 基本信息 (热度、标签、分类)
├─ 🔍 背景来龙去脉 (基于百度API搜索结果)
│  ├─ 搜索来源
│  ├─ 事件详情
│  ├─ 时间线
│  └─ 社会影响
├─ 💡 深度分析
│  ├─ 用户痛点
│  ├─ 市场机会
│  └─ 创新方向
└─ 🎯 软件产品创意 (2个方案)
   ├─ 产品1: [数据分析平台]
   └─ 产品2: [内容创作助手]
```

## 使用示例
```bash
/weibo-analysis
请分析最新的微博热搜
微博热搜数据调研
```

## 技术实现
- **Python 3.x** - 主要开发语言
- **requests** - HTTP请求库
- **json** - 数据处理
- **百度千帆AI搜索API** - 背景调研
- **天API微博热搜接口** - 数据源
- **HTML模板** - 报告生成

## 质量保证
✅ 每个话题调用百度API搜索背景
✅ 所有背景信息基于真实搜索结果
✅ 禁止虚构事件细节
✅ 每个话题提供2个软件产品创意
✅ 所有信息标注来源链接

## 注意事项
⚠️ API调用频率限制 - 程序已内置延迟机制
⚠️ 网络依赖 - 需要稳定的网络连接
⚠️ 报告生成时间 - 约30-60秒（取决于话题数量）
